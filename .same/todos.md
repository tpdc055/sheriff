## Demo Flow for Registrar Presentation

2. Dashboard shows KPIs, bottlenecks, trends
3. Cases module - create new cases, view details with timeline
4. Scheduling - view/create hearings in day/week view
5. Orders - create draft orders, issue them
6. Enforcement - log service attempts, view seizures, fee ledger
7. Audit Log - view all system activity with filters
8. Reports - various analytics views (Overview, Cases, Enforcement, Finance)
9. Reset Demo Data button to clear and reset
